## 페이지 핵심

- **히어로** — "시청이 자산이 되는 플랫폼, 그 인프라를 소유하세요" + 폰 목업(숏폼 플레이어 · XONT 채굴 배지) + 실시간 티커
- **콘텐츠** — Netflix식 TOP 10 / 신작 숏폼 드라마 포스터 레일 (CSS/SVG 목업, 한글 타이틀 선명)
- **노드/수익구조** — 노드 1개 $1,000 · 매출 10% 노드풀 · 가입자 구간별 수익 표 · CPM 차트
- **스테이킹/추천/로드맵** — APY 6/10/15/24% · 2단계 추천 · 2026.9 → 1억 유저 → 나스닥
- **대시보드 (실시간 데모)**
  - 플랫폼: 전체 노드 접속자 · 동시 스트리밍 · 총 콘텐츠 · 구독자 · 광고주 · 누적 광고수익
  - **토큰 채굴 — TONX → XONT 미러링**: 시청자가 시청 중 채굴하는 **TONX(총발행 100억)** 수량과 **1:1 동일하게** 노드 풀에 **XONT** 발행 → 전체 노드 수로 **균등 분배(÷N)**. 노드 1개 채굴 = `오늘 시청자 채굴 TONX ÷ 전체 노드 수`, 내 채굴 = `노드 1개 × 내 노드 수`. 시청자↑ → 노드 배분 자동 증가. TONX⇄XONT 스왑은 **비율 추후 공개(TBD)**
  - 내 노드 수익(1차 3,000개 풀) 실시간 배분 · 노드별 XONT 채굴 현황(균등 속도) · 현재 송출 중 콘텐츠
  - **추천 네트워크(2단계)**: 추천 코드/링크 복사 · 1단계 직접 10% · 2단계 간접 3% · 추천 인원/노드수/실시간 보상 · 추천한 사업자별 적립 현황
  - **XONT 지갑 · 스테이킹 · 다른 지갑 전송** (실동작)
- **입금 모달** — 신청 → 수량 선택 → 총액(USDT) → **TRC20 QR + 주소 복사** → 완료

## ⚠️ 운영 전 반드시 교체

1. **입금 주소** — 현재 값은 **데모 플레이스홀더**입니다.
   - `app.js`의 `DEPOSIT_ADDRESS`, `index.html`의 `#dep-addr` 텍스트를 실제 트레저리 TRC20 주소로 교체
   - QR 재생성: `npx qrcode -t svg -o assets/usdt-trc20-qr.svg "<실제주소>"`
2. **대시보드 데이터** — 현재는 데모 시뮬레이션. 2026.9 정식 가동 시 온체인 실데이터 API로 교체.
3. **면책** — 본 페이지는 정보 제공용이며 투자 권유가 아닙니다 (리스크 고지 섹션 유지).

## 비주얼 시스템

- **로고** — 커스텀 SVG 마크(레드 스퀘어클 + 플레이=노드 메시) + `TonTV` 워드마크. `index.html` 상단 `#logo-mark` 심볼.
- **아이콘** — 이모지 0개. 단일 라인 아이콘 패밀리(24px, stroke 2) 스프라이트 `#i-*`. (UI/UX 스킬 `no-emoji-icons` 준수)
- **영화 스틸컷** — `app.js`의 `scene(key, uid)`가 제목별 시네마틱 필름 스틸(레이어드 조명·보케·렌즈 플레어·실루엣·컬러그레이딩)을 **순수 SVG**로 생성. 16개 작품 + 히어로 폰 목업 + 배경 포스터 월에 적용.
  - 실제 AI 사진으로 교체하려면: 이미지 생성 후 `.poster .art` / `#ps-scene` 내부 SVG를 `<img>`로 대체하면 됩니다(구조 호환). 현재는 이미지 생성 크레딧 0이라 벡터 스틸로 제작.
- **고도화** — 스크롤 진행바, 포스터 3D 호버/포커스 링, 필름 그레인, 히어로 스태거드 로드인, Clash Display(라틴/숫자)+Pretendard(한글) 페어링.
